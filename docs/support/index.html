帮助与支持

这里是“哔哩哔哩历史记录保存与分析”的官方帮助页面。本扩展是非官方工具，与哔哩哔哩不存在隶属、授权或背书关系。

使用前检查

  1. 先在同一个浏览器中登录哔哩哔哩网页版。
  2. 点击扩展图标，再点击“保存历史记录”。首次使用建议选择“重新保存全部历史”。
  3. 保存完成后打开“记录与分析”，即可搜索、分析或导出记录。

常见问题

记录数量不完整：再次点击“保存历史记录”，并选择“重新保存全部历史”。

日期和实际不一致：在设置中把时区改为你平时使用的时区，然后刷新页面。

云端连接返回 401：检查扩展中的同步密钥是否与 Worker 的 SYNC_TOKEN 完全一致。直接用浏览器打开 Worker 地址看到 401 属于正常现象。

云端连接返回 500：检查 KV 绑定变量是否准确命名为 BILI_HISTORY_SYNC。

备份与删除

在设置中可以导出 JSON 备份。卸载扩展前建议先导出；浏览器卸载扩展时通常会删除本机扩展数据。

不再使用云端同步时，先在设置中点击“删除云端记录”，再点击“停用云端同步”。这不会删除本机历史记录。
