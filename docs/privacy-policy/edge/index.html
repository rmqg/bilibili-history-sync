Microsoft Edge 版隐私政策

生效日期：2026 年 7 月 10 日

“哔哩哔哩历史记录保存与分析”是一款适用于 Microsoft Edge 的非官方扩展，与哔哩哔哩及 Microsoft 不存在隶属、授权或背书关系。

处理的数据和目的

扩展读取当前登录用户的哔哩哔哩历史记录，用于在 Microsoft Edge 本地长期保存、搜索、备份、导出和分析。数据可能包含视频标题、BV/AV 号、UP 主、分区、观看时间、视频时长、观看进度、收藏状态和内容链接。

扩展使用 Cookie 权限检查哔哩哔哩登录状态。登录 Cookie 不会写入历史记录数据库、导出文件或云端同步内容，也不会发送给开发者。

本地保存

历史记录默认只保存在 Microsoft Edge 为本扩展提供的本地 IndexedDB 中，设置保存在扩展本地存储中。扩展不包含开发者运营的统计、广告或遥测服务，不出售数据，也不用于广告画像或信用评估。

可选云端同步

云端同步默认关闭。只有用户填写自己控制的 HTTPS 地址和同步密钥，并主动测试或同步时，扩展才会发送历史记录、观看进度、同步密钥、设备随机标识和更新时间。数据直接发送到用户指定的服务，不经过开发者服务器。

云端数据当前以明文 JSON 保存。用户可在设置页点击“删除云端记录”删除 history.json，并点击“停用云端同步”清除 Microsoft Edge 本地保存的地址、密钥和设备随机标识。本机历史记录不会因此被删除。

保留、安全和用户控制

本地数据会保留到用户清除扩展数据或卸载扩展为止。云端数据由用户自行选择的服务保留。扩展仅接受 HTTPS 云端地址；用户应保护备份文件和同步密钥。
