隐私政策

生效日期：2026 年 7 月 10 日

“哔哩哔哩历史记录保存与分析”（以下简称“本扩展”）是一款非官方浏览器扩展。本扩展与哔哩哔哩、上海宽娱数码科技有限公司及其关联公司不存在隶属、授权或背书关系。

1. 本扩展处理的数据

为了保存和分析历史记录，本扩展会处理视频标题、BV/AV 号、UP 主、分区、观看时间、视频时长、观看进度、收藏状态和内容链接。本扩展还会在浏览器本地保存自动保存间隔、时区、界面主题，以及用户主动配置的云端地址和同步密钥。

2. 登录信息与哔哩哔哩接口

本扩展使用浏览器的 Cookie 权限检查当前浏览器是否已登录哔哩哔哩，并通过哔哩哔哩官方历史记录接口读取数据。登录 Cookie 只用于完成用户请求的接口访问，不会写入历史记录数据库，不会包含在导出文件中，不会上传到用户配置的云端，也不会发送给扩展开发者。

本扩展不会删除或修改用户的哔哩哔哩账号数据。

3. 本地存储

默认情况下，历史记录和分析数据仅保存在当前浏览器的 IndexedDB 中，设置保存在浏览器扩展存储中。本扩展不接入开发者运营的统计、广告或遥测服务，也不会出售、出租或用于广告画像。

数据会保留到用户清除扩展数据或卸载扩展为止。

4. 可选云端同步

云端同步默认关闭。只有用户主动填写自己的 HTTPS 云端地址和同步密钥，并点击“测试连接”或“立即同步”时，本扩展才会连接该地址。

使用云端同步时，本扩展会向用户指定的地址发送已保存的历史记录和观看进度、用于验证请求的同步密钥、设备随机标识和更新时间。这些数据不会经过扩展开发者的服务器。用户选择的云服务提供商会按照其自身条款处理数据。

本扩展当前以明文 JSON 形式保存云端历史记录，因此只应连接用户本人控制并信任的服务。要删除已经上传的数据，请在用户自己的云存储中删除 history.json，或者删除对应的 KV 命名空间、Worker 或云端项目。

设置页提供“删除云端记录”和“停用云端同步”按钮。前者会请求用户配置的云端删除 history.json，后者会清除当前浏览器保存的云端地址、同步密钥、设备随机标识和上次同步时间；两项操作都不会删除本机历史记录。

5. 数据共享与用途限制

本扩展只为保存、查找、备份、导出、恢复、分析和用户主动发起的云端同步处理数据。除哔哩哔哩官方接口及用户自行配置的云端服务外，本扩展不会向其他第三方传输历史记录。

本扩展不会出售用户数据，不会将数据用于信用评估、广告投放或与扩展核心功能无关的用途。

6. 安全

本扩展仅允许使用 HTTPS 云端地址，并建议使用独立、足够长的同步密钥。用户应妥善保护浏览器配置、备份文件和同步密钥。

7. 政策变更

如果数据处理方式发生实质变化，本政策会更新生效日期，并在发布说明或扩展界面中说明重要变化。

8. 联系方式

帮助与支持

问题、隐私请求或安全问题可通过 GitHub Issues 联系。
