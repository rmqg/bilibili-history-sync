哔哩哔哩历史记录保存与分析

把哔哩哔哩历史记录保存在自己的浏览器中，长期查找、备份，并查看观看进度、视频时长、分区和 UP 主分析。

Safari

查看免费安装教程

Chrome、Edge 和 Firefox 审核通过后，本页会更新为直接安装链接。Safari 免费安装不需要购买 Apple Developer Program。Release 中的浏览器 ZIP 面向开发者模式安装，不是商店安装程序。

数据由你保管

历史记录默认只保存在当前浏览器。扩展不会删除或修改哔哩哔哩账号中的记录；只有你主动配置并使用云端同步时，数据才会发送到你指定的云端地址。
